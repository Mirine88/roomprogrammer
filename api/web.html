요청 (GET)

roomprogrammerapi.herokuapp.com/web?language=${language}&paths=${paths}&values=${values}


매개변수 설명 필수 비고
language 얻을 코드의 언어입니다. 필수 js, go가 지원됨
paths 경로입니다. 필수 ','로 구분하여 입력
슬러시('/')를 포함해야합니다.
values 경로에 따른 화면에 나오는 값입니다. 필수 ','로 구분하여 입력
paths와 길이가 같아야함
